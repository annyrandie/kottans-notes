# linux cli notes

**ls**

    lists the contents of your current directory

**mkdir**

    create a directory

**mv file1 file2**
  
    move <file1> to <file2>
    
**cd directory**

    change directory
    
**print working directory**

    pwd
    
**cp directory directory-copy**

    copy directory directory-copy
    
**rm**

    remove file
    
**rmdir**

    remove directory
